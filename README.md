# MangaGO

Plinio e Henrique

Gestão de Acervo e Inventário Pessoal de Colecionáveis.


O projeto MangaGO consiste em uma plataforma de gestão de coleções de mangás desenvolvida em Python com o framework FastAPI. O sistema funciona como um assistente digital para colecionadores e leitores de quadrinhos japoneses que desejam organizar sua biblioteca física ou digital de forma prática. Através dele, o usuário consegue cadastrar novos títulos, controlar a quantidade de volumes que possui em sua estante e atualizar o status de leitura de cada obra. O objetivo principal é resolver o problema comum de desorganização em grandes acervos, evitando que o colecionador compre volumes repetidos por esquecimento ou perca o controle de em qual capítulo ou volume interrompeu sua leitura.
